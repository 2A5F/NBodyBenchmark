# NBodyBenchmark

一个用 unity ecs 写的 n 体计算基准测试

以下是使用 7950x 运行一万个天体的帧率

![image](https://user-images.githubusercontent.com/13982338/217540428-c051f6e1-e95a-4507-99c8-eb7c855e6028.png)

### 操作

- `esc`  
  菜单
- `space`  
  暂停
- `1`, `2`, `3`, `4`, `5`  
  调速度
- `w`, `s`, `a`, `d`  
  前后左右
- `e`, `q`  
  上下
- `鼠标按住右键拖动`  
  旋转视角
- `鼠标滚轮`  
  调整移动速度 
- `shift`  
  临时加速移动
