### 操作

- `esc`  
  菜单
- `space`  
  暂停
- `1`, `2`, `3`, `4`, `5`  
  调速度
- `w`, `s`, `a`, `d`  
  前后左右
- `e`, `q`  
  上下
- `鼠标按住右键拖动`  
  旋转视角
- `鼠标滚轮`  
  调整移动速度 
- `shift`  
  临时加速移动
